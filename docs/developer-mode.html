開発者モード

⚠️ 注意: 開発者モードは実験的な機能であり、正常に動作しない可能性があります。また、将来のバージョンで変更や削除される可能性があります。使用する際は留意をお願いします。

開発者モードの有効化

開発者モードを有効にすると、以下の追加機能が利用できます：

  • JSONファイルから動画リストのインポート
  • YouTubeリストからCSVのダウンロード
  • 音声重視モード

有効化方法

  1. アプリのタイトル「Uta(Gawa)Picker」を5回連続でタップ（クリック）
  2. 「開発者モードが有効になりました」というメッセージが表示されます
  3. アプリバーの右上に開発者モード用のボタンが表示されます

ヒント: タイトルをタップしてから2秒以内に次のタップを行わないと、カウントがリセットされます。素早く5回タップしてください。

JSONからプレイリストを作成

開発者モードが有効になると、JSONファイルから動画リストを一括インポートできます。

使用方法

  1. アプリバーの右上にある「📤 JSONからプレイリストを作成」ボタンをクリック
  2. JSONファイルを選択
  3. 動画リストが表示されます
  4. 各動画の「➕ プレイリストに追加」ボタンをクリックして、プレイリストに追加
  5. ダイアログが表示されます。開始時刻、終了時刻、楽曲タイトルを入力できます（任意項目）
  6. 「追加」ボタンをクリックすることで、プレイリストに反映されます

JSONファイルの形式

JSONファイルは以下の形式である必要があります：

[ { "videoId": "dQw4w9WgXcQ", "title": "動画タイトル" }, { "videoId": "abc123def45", "title": "別の動画タイトル" } ]

注意: JSONファイルは、uta_picker/tree/main/scriptのfetch_videos.pyスクリプトを使用して生成できます。詳細はREADMEに記載されています。

💡 便利な機能: これらのPythonスクリプトは、Google Colab形式のノートブックとして提供されており、自身の環境で簡単に実行できます。Google Colabで開いて、そのまま実行することが可能です。

YouTubeリストからCSVをダウンロード

開発者モードが有効になると、YouTubeプレイリストIDを指定してCSVファイルを直接ダウンロードできます。

使用方法

  1. アプリバーの右上にある「📥 YouTubeリストからCSVをダウンロード」ボタンをクリック
  2. YouTubeプレイリストIDを入力（PLから始まるID。例：PLw6tV8gK-20tVH2w-ZK-SAZ5lErzMLrj5）
  3. 「ダウンロード」をクリック
  4. CSVファイルがダウンロードされます

💡 便利な機能: CSVファイルを生成するPythonスクリプト（fetch_comments.py）は、Google Colab形式のノートブックとして提供されており、自身の環境で簡単に実行できます。Google Colabで開いて、そのまま実行することが可能です。

音声重視モード

⚠️ 注意: 音声重視モードは現在正常に動作していないことが確認されています。この機能を使用しても期待通りの動作をしない可能性があります。

開発者モードが有効になると、音声重視モード（低解像度）を有効にできます。これにより、データ使用量を抑えながら音声を重視した再生が可能になります。

使用方法

  1. アプリバーの右上にある「🎵 音声重視モード」ボタンをクリック
  2. モードが切り替わります（ボタンの色が青くなります）
  3. 再度クリックすると通常モードに戻ります

開発者モードを無効にする方法

注意: 現在のバージョンでは、開発者モードを無効にする機能はありません。ページを再読み込みしても開発者モードは維持されます。完全に無効にするには、ブラウザのキャッシュをクリアする必要があります。
